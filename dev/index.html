Introduction

CImGui

This package provides a Julia language wrapper for cimgui: a thin c-api wrapper programmatically generated for the excellent C++ immediate mode gui Dear ImGui. Dear ImGui is mainly for creating content creation tools and visualization or debug tools. Browse Gallery to get an idea of its use cases.

Installation

pkg> add CImGui
